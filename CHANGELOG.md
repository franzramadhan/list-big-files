## [0.1.0] - 2025-01-07

### Added
- Initial release of List Big Files
- Parallel file scanning using Rayon
- Support for recursive directory traversal
- Flexible size specification with MB and GB units
- Built-in help system with `--help` and `help` commands
- Performance timing information
- Sorted output (largest files first)
- Comprehensive documentation (README, CONTRIBUTING, LICENSE)
- Build script for easy compilation

### Features
- Fast parallel processing of file metadata
- Support for multiple size units: MB, M, GB, G
- Default size threshold of 100MB
- Formatted table output with file sizes and paths
- Error handling for permission issues
- Cross-platform support (macOS, Linux, Windows)

### Documentation
- Complete README with installation and usage instructions

- MIT License
- Inline code comments explaining logic
